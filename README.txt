1) First open the EXE file this is to install python, if you already have python 3.12.2 or above the skip to step two.
    -When installing python Make sure that the button to add python to PATH is checked.
    -Then Click customise installation
    -Then make sure that all buttons are checked
    -Click next
    -Tick the first five boxes but nothing else, ignore the path below
    -Click install then Yes
    -Finally click the 'Disable Path Length Limit' button, yes, then close
2) Second, open up the setup file (.cmd). This installs the required modules if needed. Run this whether or not you think they are installed.
3) Then you should be ready to go. Open up the run.cmd
